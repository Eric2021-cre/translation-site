## Before / After：AI翻译 → 人工修改

### 样例①：商务邮件（中文 → 日文）

**原文（简体中文）：**
> 您好！感谢您在百忙之中抽出时间来参加我们的会议。关于您提出的方案，我们已经进行了详细的讨论，总体来说方向是可以接受的，但有几个细节需要进一步确认。

**AI翻译（修改前）：**
> こんにちは！お忙しい中、私たちの会議に参加していただきありがとうございます。…（直译・生硬）

**人工修改后：**
> お世話になっております。ご多用の折にもかかわらず、会議にご参加いただきありがとうございます。…（自然・商务风格）
